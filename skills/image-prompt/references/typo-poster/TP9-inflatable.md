# TP9 인플레이터블 타입 (누르고 싶은 글자)

- 성격: 글자가 공기 주입된 비닐 풍선 — 팽팽한 표면장력, 봉제선, 주입구 밸브까지 갖춘 **진짜 풍선의 증거**가 있어야 성립한다. 감탄보다 호감을 만드는 촉각형 쇼스타퍼.
- 관찰 시그니처:
  - **풍선 증거 필수 2종+**: 봉제선 `visible heat-sealed seams running along each letter edge` · 주입구 `a tiny inflation valve tucked at the base of one letter` · 눌림 `letters squishing against each other where they touch`
  - 팽팽한 표면장력 + 큰 글로스 하이라이트가 부드럽게 번짐
  - 파스텔 그라데이션 또는 단색 새틴 비닐 마감
  - 미니멀 단색 배경, 바닥 소프트 섀도(또는 공중 부유 + 아래 섀도)
- 드롭인: `the word "{단어}" as inflated vinyl balloon letters packed tightly into the frame, visible heat-sealed seams running along each letter edge, a tiny inflation valve tucked at the base of one letter, letters squishing softly against each other, broad glossy highlights on the taut surfaces, soft shadow beneath`
- 팔레트 시작값: 배경 크림 `#F3EDE4` · 비닐 파스텔 2 (`#F4A8C1` 핑크 · `#9EC8F0` 블루)
- 주의: 풍선 증거를 명시하지 않으면 그냥 둥근 3D 글자로 퇴행 — 봉제선·밸브 중 2개 이상 반드시. 글자끼리 눌리는 squish가 귀여움의 핵심. **한글 1~2자**(예: "폭신"은 2컷 분할보다 2자 한 컷). `AR 4:5` / `1:1`
